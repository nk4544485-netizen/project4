# PhonePe Pulse Dashboard

A multi-page Streamlit application designed for PhonePe Pulse data analysis.

## Structure
- \`app.py\`
- \`requirements.txt\`
- \`README.md\`
- \`.streamlit/config.toml\`
- \`pages/\`
  - \`1_Geography.py\`
  - \`2_Transactions.py\`
  - \`3_Users.py\`
  - \`4_Top_Performers.py\`
- \`data/\`
  - \`phonepe_pulse.db\`
- \`create_db.py\`

## Run the app
\`\`\`bash
streamlit run app.py
\`\`\`
